# Atlantic IT Support

Atlantic IT Support provides technical support services including device troubleshooting, networking setup, malware removal, and smart home configuration for individuals and small businesses.

## Services

- PC & Laptop troubleshooting and setup
- Wi-Fi & Networking (router setup, dead zones, dropouts)
- Smart Home / IoT device setup
- Malware cleanup
- Email & Cloud support (Gmail, Outlook, M365)

## CompTIA AI Quiz Generator

Free, AI-powered practice quizzes for CompTIA certifications:

- **CompTIA A+** (220-1201 & 220-1202)
- **CompTIA Network+** (N10-009)
- **CompTIA Security+** (SY0-701)

Questions are generated by Google Gemini AI, so every quiz is unique. Includes instant feedback and explanations.

## Tech Stack

- Static HTML, CSS, and vanilla JavaScript
- Netlify Functions (serverless) for Gemini API proxy
- Hosted on Netlify

## Project Structure

```
.
├── index.html                  # Main services page
├── comptia-guidance.html       # AI quiz generator
├── remote-support.html         # Remote support page
├── about.html                  # About page
├── styles.css                  # All styles
├── script.js                   # Shared scripts (nav, forms, footer)
├── quiz.js                     # Quiz engine (calls Gemini via Netlify function)
├── netlify.toml                # Netlify config
└── netlify/
    └── functions/
        └── generate-quiz.js    # Serverless proxy for Gemini API
```

## Setup

1. Clone the repo
2. Set `GEMINI_API_KEY` as an environment variable in Netlify
3. Deploy via Netlify (linked to this repo)
